flask SQLAlchemy 的安装和基础应用

2018-12-23

SQLAlchemy是一个基于Python实现的ORM框架。该框架建立在 DB API之上，使用关系对象映射进行数据库操作，简言之便是：将类和对象转换成SQL，然后使用数据API执行SQL并获取执行结果。 安装命令pip install flask-sqlalchemy 注意sqlalchemy 依赖于 pymysql 模块，确保pymysql 被正确安装 pip install pymysql 建立对象#导入第三方连接库sql点金术 from flask_sqlalchemy import SQLAlchemy #建立对象 app = Flask(__name__) #载入配置文件 app.config.from_pyfile(&#39;config.ini&#39;) # #指定数据库连接还有库名..
